# 🎂 Bolo Delicioso

> E-commerce de confeitaria artesanal — projeto de conclusão de curso

---

## 📖 Sobre o Projeto

O **Bolo Delicioso** é um site de e-commerce voltado para a venda de bolos e doces artesanais. Desenvolvido como projeto de conclusão de curso, o site conta com uma interface amigável, navegação intuitiva e layout responsivo utilizando Bootstrap.

🔗 **Acesse o site:** [adrianolis.github.io/bolo-delicioso](https://adrianolis.github.io/bolo-delicioso/)

---

## 📄 Páginas

| Página | Arquivo | Descrição |
|---|---|---|
| Página Inicial | `index.html` | Carousel de banners e informações gerais |
| Promoções | `promoções.html` | Produtos com desconto |
| Mais Vendidos | `mais-vendidos.html` | Grade com os 10 bolos mais vendidos |
| Login | `login.html` | Acesso à conta do cliente |
| Cadastro | `cadastro.html` | Criação de nova conta |
| Recuperação de Senha | `recuperação.html` | Formulário para redefinir senha |
| Carrinho | `carrinho.html` | Página de compras do cliente |
| Contatos | `contatos.html` | Informações de contato e redes sociais |

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- [Bootstrap 4.5](https://getbootstrap.com/docs/4.5/)
- jQuery 3.5
